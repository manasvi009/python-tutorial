# Jarvis Virtual Assistant - Fully Functional!

## ✅ FIXED: Now Works Continuously!
Your Jarvis assistant will now keep running after each command instead of stopping after one task.

## 🚀 NEW: App Launching Capability
You can now open applications with voice commands!

### Supported Apps:
- **Microsoft Office**: word, excel, powerpoint
- **System Tools**: notepad, calculator, paint, cmd, powershell, task manager
- **Browsers**: chrome, firefox, edge
- **Media**: vlc, spotify, steam
- **Dev Tools**: vs code, sublime
- **And many more!**

### Command Examples:
- "Jarvis, open notepad"
- "Jarvis, launch calculator" 
- "Jarvis, start chrome"
- "Jarvis, open word app"
- "Jarvis, launch discord"

### Other Features Still Work:
- Open websites: "open google", "open youtube"
- Play music: "play stealth"
- Get news: "news" (requires API key)
- Chat with AI: "How are you?" (requires API key)

## How to Use:
1. Run: `python main.py` or `python jarvis_basic.py`
2. Say: "Jarvis" to wake the assistant
3. Give your command (e.g., "open notepad")
4. The assistant will respond and continue listening for more commands
5. Say "bye" to exit

## Two Versions Available:
- **main.py**: Full version with AI features (requires API keys)
- **jarvis_basic.py**: Basic version without API requirements

Both versions will now handle multiple requests continuously! 🎉
